    Link to this function search_by_query(type, query, opts \\ %{})

    Fetch a list of the videos or playlists based on type for the search query.

    If the returned items are videos they are represented as an IO list that contains a title, an id, a channel title, a publish date and a description.

    If the returned items are playlists they are represented as an IO list that contains a title, an id, a channel title and a publish date.
